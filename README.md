# GenomicTools

## Input formats:

Species data (comma separated):
		Row 1: # Genus_species
		Row 2: # column labels
		- Col. 1: chromosome name
    - Col. 2: chromosome number
		- Col. 3: relative index
    - Col. 4: absolute index
		- Col. 5: orthogroup 
		- Col. 6: gene name
		- Col. 7: MAKER gene label
		- Col. 8: empty
    - Col. 9: empty
    - Col. 10: empty
    - Col. 11: empty
    - Col. 12: empty
    
Dot plot files (comma separated):
    Row 1: # Genus_species A, Genus_species B
    Row 2: # column labels
		- Col. 1: chromosome name A
		- Col. 2: relative index A
    - Col. 3: chromosome name B
    - Col. 4: relative index B
		- Col. 5: empty
		- Col. 6: empty
    - Col. 7: empty
		- Col. 8: empty
		- Col. 9: empty
